NAnt TeamCity Logger

Download Source Download Binary View on GitHub

What is this?

This is a logger that you can use with NAnt so that TeamCity will properly display log messages from NAnt.

Installation

Download the latest version from our SourceForge site.

Extract the file Boinst.NAntExtensions.TeamCity.dll from the archive and copy it into the extensions\common\4.0 subdirectory of your NAnt installation.

Invokation

Launch NAnt with the following syntax:

NAnt.exe -logger:Boinst.NAntExtensions.TeamCity.TeamCityLogger

Problems?

Please report any issues on our GitHub issue tracker.
